Элемент picture — адаптирует картинки под разные устройства и поддерживаемые форматы

Измените размер браузера, чтобы увидеть разные версии загрузки изображений с разными разрешениями. Сначала браузер ищет первый элемент тега source, где медиа-запрос соответствует ширине текущего окна просмотра браузера, и извлекает изображение, указанное в атрибуте srcset.

Элемент img нужен в качестве последнего тега блока. Используется для обеспечения обратной совместимости для браузеров, которые не поддерживают или если ни один элемент из тегов source не подходит под условия.
