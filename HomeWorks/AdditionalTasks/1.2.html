Теги для оформления текста

Теги моноширинного шрифта: pre, code, kbd.

  • Тег ‹tt› отображает текст моноширинным шрифтом.
  • Тег ‹pre› отображает текст в браузере 
     со   вcеми пробелами     между словами
    (по умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один).
  • Тег ‹code› отображает текст, являющийся программным кодом: style="color: blue; text-align: center"
  • Тег ‹kbd› обозначает пользоватальский ввод: с клавиатуры, голосом или другим устройством ввода информации, или для названия клавиш - Ctrl+S

Теги для создания списка определений: dl, dt, dd.

C тега ‹dl› начинается каждый такой список, куда входит тег ‹dt›, задающий термин, а тег ‹dd› - определения данного термина.
Пример.
HTML
Стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере.
CSS
Язык разметки, используемый для визуального оформления веб-сайтов.

Другие теги

  • Тег ‹sup› служит для отображения текста в виде верхнего индекса .
  • Тег ‹sub› служит для отображения текста в виде нижнего индекса .
  • Тег ‹q› служит для выделения в тексте цитат, заключенных в кавычках. Например: В хорошем дизайне добавление чего-то стоит дешевле, чем сама эта вещь. Thomas C. Gale.
  • Тег ‹blockquote› предназначен для выделения длинных цитат внутри документа. Например:

    Для меня долгое время было загадкой, как что-то очень дорогое и технологичное может быть столь бесполезным. И вскоре я осознал, что компьютер — это глупая машина, обладающая способностями выполнять невероятно умные вещи, тогда как программисты — это умные люди, у которых талант делать невероятные глупости. Короче, они нашли друг друга.

  • Тег ‹var› служит для выделения переменных (x = 5a - b).
  • Тег ‹del› заставляет используется для выделения текста, который был удалён в новой версии документа, а тег ‹ins› предназначен для выделения текста, который был добавлен в новую версию документа.
Тег ‹nobr› заставляет отображать текcт без переносов, одной строкой, а тег ‹wbr› указывает место, где допускается перенос строкив тексте.

Тег ‹address› предназначен для хранения информации об авторе:

Все это собственноручно создано Лаптевой Дарьей!!! Ладно...Разрешаю смотреть.
